## Run Project

Dependencies: [Node.js](https://nodejs.org/en/), [Yarn](https://yarnpkg.com);

Fist make a fork/clone of this repo and then on the folder:

```sh
yarn install
```

To run locally:

```sh
yarn dev
```

Build for production:

```sh
yarn build
```

Preview production version _(require run build first)_:

```sh
yarn preview
```
